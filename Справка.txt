								     Пользовательская справка

									Работа с файлами

1. Для работы с файлами выберете пункт "Файл" в основном меню. В раскрывшемся списке вы уввидите следующие пункты:
	Открыть - непосредственное открытие файла
	Сохранить - для сохранения изменений в текущем файле
	Сохранить как - для сохранения данных в новом файле

*Также эти компоненты вызываются на панели инструментов (см. всплывающие подсказки на компонентах панели)
2. При первоначальном запуске пункты "Сохранить" и "Сохранить как" недоступны. Данные кнопки разблокируются при добавлении записи либо открытии файла.
3. В пункте меню "Файл" также есть пункт "Выход". При подтверждении всех всплывающих сообщений при нажатии на него, будьте готовы к тому, что даннные запи-
си либо изменения будут утеряны!


								     Правка данных таблицы

1. Все способы правки записей в таблице находятся в пункте основного меню "Правка":
	Добавить - добавление новой записи. Для успешного добавления записи убедитесь в корректном и полном заполнении полей всплывающего диалогого окна.
	Редактировать - редактирование существующей записи. Для редактирования выделите любую ячейку внутри строки, в которую вы хотели бы внести изменения.
	Удалить - удаление существующей записи. Для удаления выделите любую ячейку внутри строки, в которую вы хотели бы удалить.

2. Также компоненты правки вызываются щелчком правой кнопки мыши внутри табличного пространства.
3. Также компоненты правки находятся на панели инструментов (см. всплывающие подсказки на компонентах панели)


								  Обработка данных таблицы

Все способы обработки записей в таблице находятся в пункте основного меню "Обработка":
	Поиск - позволяет находить записи по словам или сочетанию символов внутри какого-либо поля, заданного пользователем. 
		Поиск может осуществляться посредством основного меню, либо на панели инструментов (см. всплывающие подсказки на компонентах панели). 
			1)Выберете поле для поиска
			2)Далее всплывает диалоговое окно для ввода поисковой инфрмации. 
			3)Введите запрос и нажните "ОК"	
			4)В появившемя окне содержится таблица с найденными данными по запросу
		Поиск может также осуществляться с помощью пустого поля и кнопки "Поиск" в правой части формы. 
			1)Для поиска введите запрос в поле
			2) При нажатии на кнопку "Поиск" выберете поле поиска
			3)В появившемя окне содержится таблица с найденными данными по запросу
	Сортировка - сортировка от А до Я, от Я до А, либо по убыванию и возрастанию для поля "Дата рождения" записей таблицы. 
		Сортировка находится в основном меню и в компонентах панели инструментов (см. всплывающие подсказки на компонентах панели)
	Родословная - позволяет составить родословную заданной собаки, которая отображается в всплывающем окне.
		Компонент находится в основном меню, компонентах панели инструментов (см. всплывающие подсказки на компонентах панели) либо находится в кон-
		текстном меню, вызываемом щелчком правой кнопки мыши.
		Для составления родословной собаки выделите строку, содержащую информацию о ней.
	Самая популярная порода - в виде вообщения выводится информация о самой популярной породе среди внесенных. Компонент находится в основном меню.
	Самая непопулярная порода - в виде вообщения выводится информация о самой непопулярной породе среди внесенных. Компонент находится в основном меню.
	Владельцы 2 и более собак - сопровождается открытием окна, содержащего таблицу и ФИО владельцев, у которых больше 1 собаки, и количеством их собак 
		соответственно. Компонент находится в основном меню.
	Список по породе - сопровождается открытием окна, содержащего записи о всех внесенных в основную таблицу собаках породы, выбранной пользователем. 
		Компонент находится в основном меню.
		При нажатии на компонент выпадает список пород для выбора.


								 Информация о программе

Всю информацию о программе можно найти в пунке основного меню "О программе":
	Пункт "Задание" содержит постановку задачи
	Пункт справка открывает текущее окно, содержащее справочеую информацию. Также справка вызывается на панели инструментов (см. всплывающие подсказки 
		на компонентах панели)
	Пункт "О программе" содержит информацию о последних изменениях в программе и контакты автора.	
	

								     Темы оформления
Изменить цветовое оформление приложения можно на вкладке основного меню "Темы оформления".
	Вы можете выбрать цвет из 5 предложенных цветов: светло-серый,серый, светло-голубой, голубой, зеленый. По умолчанию в приложении установлен светло-
серый цвет.
